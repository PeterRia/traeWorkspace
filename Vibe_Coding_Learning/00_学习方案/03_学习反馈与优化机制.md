# 学习反馈与优化机制

## 反馈循环

```
学习 → 实践 → 评估 → 反思 → 调整 → 学习
  ↑                                    ↓
  └──────────── 持续改进 ←──────────────┘
```

## 日常反馈（每日）

- [ ] 今天学习了什么知识点？

## 阶段性反馈（每个知识点完成后）

### 知识点掌握度评估
| 评估维度 | 完全不懂(1) | 知道概念(2) | 能模仿(3) | 能独立完成(4) | 能教别人(5) |
|---------|:---------:|:---------:|:-------:|:-----------:|:---------:|
| 理论理解 | | | | | |
| 动手实践 | | | | | |
| 问题解决 | | | | | |

### 调整策略
- **得分 < 3 分**：重新学习，补充基础，增加练习量
- **得分 3-4 分**：正常推进，记录薄弱环节
- **得分 5 分**：可以加速，尝试更高难度的挑战

## 动态调整策略

### 进度超前（提前完成）
- 深入附录内容，夯实基础
- 尝试更高难度的扩展项目
- 帮助社区其他学习者

### 进度落后（未按时完成）
- 分析卡点：是理论难懂还是实践遇坑？
- 减少每日任务量，聚焦核心
- 寻求帮助：AI 问答、社区讨论、查阅文档

### 遇到瓶颈
- 换个角度：看视频教程、阅读不同资料
- 降低难度：先做简化版本
- 休息调整：适当放松后再回来
