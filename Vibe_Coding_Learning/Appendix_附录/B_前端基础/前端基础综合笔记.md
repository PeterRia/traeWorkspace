## 02 前端框架进化史

### 演进路线
```
纯 HTML/CSS/JS → jQuery → Angular → React → Vue → Next.js/Nuxt
```

### 当前主流

| 框架 | 特点 | 适合 |
|------|------|------|
| React | 生态最大，组件化 | 大型项目 |
| Vue | 易学，中文友好 | 中小项目 |
| Next.js | React + SSR | 全栈应用 |
| Tailwind CSS | 原子化 CSS | 快速样式开发 |

### Vibe Coding 推荐
用 Vibe Coding 做前端时，推荐使用 React + Tailwind CSS 组合，AI 对这两个工具支持最好。
