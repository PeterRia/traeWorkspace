# Vibe Coding 系统化学习项目

基于 Datawhale Easy-Vibe 官方教程 (https://datawhalechina.github.io/easy-vibe/zh-cn/) 构建的完整 Vibe Coding 学习体系。

> **Vibe Coding（氛围编程）**：通过自然语言对话与 AI 协作完成编程项目，让"会说话就会编程"成为现实。

## 学习路径

| 阶段 | 目标人群 | 核心能力 | 预计耗时 | 产出 |
|:---:|---------|---------|:---:|------|
| Stage 1 | 零基础 / 产品经理 | AI 对话编程、原型开发 | 2 周 | 可交互 Web 原型 |
| Stage 2 | 初级开发者 | 全栈开发、部署上线 | 3 周 | 完整可上线产品 |
| Stage 3 | 中级开发者 | 架构设计、多端开发 | 4 周 | 跨平台应用 + 个人品牌 |

## 快速开始

1. 阅读 `00_学习方案/01_个性化学习方案.md` 了解整体规划
2. 从 `Stage1_零基础入门/01_学习地图/` 开始第一个知识点的学习
3. 每完成一个知识点，在 `00_学习方案/02_进度跟踪表.md` 中记录进度
4. 阶段性在 `00_学习方案/03_学习反馈与优化机制.md` 中进行自我评估

## 核心理念

- **完成比完美更重要** - 先做出能跑的东西，再迭代优化
- **对话即编程** - 用自然语言描述需求，AI 帮你实现
- **项目驱动学习** - 在真实项目中掌握技能
- **痛点驱动产品** - 从用户真实痛点出发做产品
