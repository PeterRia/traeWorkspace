# 01 学习地图 - 代码示例：第一个 AI 对话编程

## 目标
了解如何使用自然语言与 AI 进行编程对话，体验 Vibe Coding。

## 示例一：让 AI 写一个简单网页

### 提示词（在 z.ai 或 ChatGPT 中输入）
```
帮我写一个简单的个人介绍网页，包含：
1. 一个标题"你好，我是[你的名字]"
2. 一段自我介绍文字
3. 一个技能列表
4. 一个联系方式区域
5. 界面简洁好看，用现代设计风格
```

### 预期效果
AI 会生成一个包含 HTML 和 CSS 的完整网页，可以直接在浏览器中打开。

---

## 示例二：让 AI 解释代码

### 场景
AI 生成了代码，但你看不懂某些部分。

### 提示词
```
请解释这段代码每一行的作用：
[粘贴 AI 生成的代码]

特别说明：
- 这段代码的整体功能是什么？
- 每个关键标签和属性的作用是什么？
```

---

## 示例三：让 AI 修复问题

### 场景
页面显示不符合预期，按钮没有居中。

### 提示词（附带截图更佳）
```
我生成的页面中，按钮没有居中显示，请帮我修复。
当前代码是：
[粘贴当前代码]

期望效果：
- 按钮在页面中间水平居中
- 按钮有圆角和阴影效果
- 鼠标悬停时有颜色变化
```

---

## 示例四：多轮对话迭代

### 第一轮
```
帮我做一个简单的待办事项列表页面。
```

### 第二轮（基于 AI 的输出）
```
很好！现在请添加以下功能：
1. 可以标记任务为"已完成"（文字加删除线）
2. 已完成的任务显示不同的背景色
3. 添加一个"清除已完成"按钮
```

### 第三轮
```
再加一个搜索框，可以筛选显示包含特定关键词的任务。
```

---

## 关键技巧总结

1. **描述要具体**：不要只说"做个好看的页面"，要说明具体的布局、颜色、功能
2. **分步骤迭代**：先做基本功能，再逐步添加高级功能
3. **遇到问题多问**：不要自己闷头排查，直接把问题描述给 AI
4. **保存提示词**：好的提示词模板可以复用，效率越来越高
