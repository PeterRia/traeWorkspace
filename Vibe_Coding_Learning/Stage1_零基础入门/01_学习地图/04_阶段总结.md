# 01 学习地图 - 阶段总结

## 核心概念回顾

### Vibe Coding 的三个核心信念
1. **会说话就会编程**：编程语言正在变成自然语言
2. **完成比完美更重**：先做出能跑的东西，再迭代
3. **执行力和想法 > 技术**：AI 时代，想法比编码能力更重要

### 成长路径
```
新手入门 → 初中级开发 → 高级开发
（原型/Demo）（全栈产品）（多端/架构）
```

### 关键技能
- **提问能力**：能清晰描述需求和问题
- **迭代思维**：逐步优化，不追求一步到位
- **工具选择**：了解不同 AI 编程平台的特点
