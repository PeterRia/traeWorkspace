# 贪吃蛇项目 - 提示词模板集合

## 基础版贪吃蛇
```
帮我做一个贪吃蛇游戏：
1. 用方向键控制蛇的移动
2. 吃到食物后蛇会变长，分数增加
3. 撞到墙壁或自己的身体就游戏结束
4. 要有开始和重新开始按钮
5. 界面要简洁好看
```

## AI 原生版（完整功能）
```
我想让 AI 生成一个网页贪吃蛇游戏，需要一个更完整的提示词，让生成结果更令人印象深刻和有趣。

目标：
- 贪吃蛇游戏需要实现吃不同单词的功能
- 收集的单词用于生成诗歌
- 诗歌完成后自动生成对应图像

请生成一个详细的提示词。
```

## 功能扩展提示词
```
在现有贪吃蛇基础上，添加以下功能：
1. 蛇吃到特殊单词时改变颜色
2. 添加"魔法单词"特效（如"风"改变背景为风的效果）
3. 每次吃单词都有粒子动画效果
4. 游戏结束后展示所有生成的诗歌和图像
```

## 设计优化提示词
```
请优化当前贪吃蛇游戏的视觉设计：
1. 使用现代简约风格
2. 配色方案：深色背景 + 霓虹色蛇身
3. 添加平滑的动画过渡效果
4. 字体使用圆体/无衬线字体
5. 游戏区域有微妙的网格背景
```

## 响应式适配提示词
```
请让当前贪吃蛇游戏支持移动端：
1. 在手机上显示虚拟方向键
2. 触摸滑动控制蛇的方向
3. 游戏区域自适应屏幕大小
```
