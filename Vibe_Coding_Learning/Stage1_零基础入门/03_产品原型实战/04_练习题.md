### 练习 2：5 步验证法实践
选一个你想做的产品方向，完成 5 步验证：
1. 找到 5 个目标用户，了解他们现在的做法
2. 分析现有替代方案的优缺点
3. 测试用户的付费意愿
4. 估算市场规模
5. 思考你的竞争壁垒

### 练习 3：横向切分练习
将以下"大而泛"的需求切分成精准的细分人群：
1. "我想做一个健身 APP" → 切分成 3 个细分人群
2. "我想做一个学习工具" → 切分成 3 个细分人群
3. "我想做一个社交产品" → 切分成 3 个细分人群

## 进阶练习（选做）

### 练习 4：产品概念重构
选择一个热门产品（如 Keep、小红书、美菜网），分析：
1. 它解决的核心痛点是什么？
2. 它的目标用户是谁？（做横向切分）
3. 用户的完整使用场景是怎样的？（纵向深挖）
4. 为什么用户愿意付费？

### 练习 5：完整原型项目
按照"实践项目：从想法到原型"的流程，完成一个完整产品原型：
1. 找到一个真实痛点
2. 访谈 5 个目标用户验证需求
3. 用 AI 完善产品设计
4. 用 Vibe Coding 开发可交互原型
5. 写一份项目复盘

### 练习 6：竞品分析
选一个产品方向，用 AI 做竞品分析：
```
请帮我分析 [产品方向] 的竞品格局：
1. 列出主要的直接竞品和间接竞品
2. 分析每个竞品的优势和劣势
3. 找出市场空白点
4. 给出差异化建议
```

## 参考答案

### 练习 1 参考
- 糖尿病患者计算碳水：痛点（威胁健康）
- AI 照片滤镜：痒点（可有可无的酷炫）
- 外卖 30 分钟送达：爽点（即时满足）
- 记录步数：痒点（想象中的健康生活）
- 凌晨起床买菜：痛点（影响生存质量）
- 手机桌面主题：痒点（美观但不必须）

### 练习 2 参考
横向切分示例：
1. 健身 APP → 产后妈妈 / 糖尿病患者 / 增肌人群 / 学生群体
2. 学习工具 → 考研学生 / 程序员 / 语言学习者 / 考证人群
3. 社交产品 → 宝妈群体 / 退休老人 / 创业者 / 兴趣小组
