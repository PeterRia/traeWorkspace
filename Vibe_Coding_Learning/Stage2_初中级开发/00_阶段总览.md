# Stage 2 · 初中级开发 - 阶段总览

## 阶段定位

从"能做出原型"到"能做出可上线的完整产品"。一个人就是一支团队——从前端到后端，从数据库到上线。

## 学习目标

- 学会使用 AI 工具生成和管理素材资源
- 掌握专业 UI 设计工具（Figma/MasterGo）的基础操作
- 实现从设计稿到前端代码的完整转换
- 使用 Supabase 构建数据驱动的应用
- 完成产品从本地开发到云端的部署上线
- 集成 Dify 构建 AI 知识库应用

## 章节结构

| 章节 | 内容 | 预计耗时 | 难度 |
|------|------|:---:|:--:|
| 01 素材生成 Agent | Lovart/Nanobanana 工作流搭建 | 4h | ⭐⭐ |
| 02 Figma 与 MasterGo | UI 设计工具入门 | 3h | ⭐⭐ |
| 03 设计稿转代码 | 设计到前端代码的转换 | 4h | ⭐⭐⭐ |
| 04 真实数据项目 | Supabase 数据库设计与使用 | 5h | ⭐⭐⭐ |
| 05 部署上线 | Vercel/Zeabur 部署全流程 | 4h | ⭐⭐⭐ |
| 06 AI 知识库集成 | Dify RAG 应用开发 | 5h | ⭐⭐⭐⭐ |

## 学习方法

1. **工具先行**：先熟悉每个工具的基本操作
2. **项目串联**：用一个项目串起所有知识点
3. **遇到问题多查文档**：这些是真实生产工具，文档完善

## 产出要求

- 素材生成工作流 + 绘图 Agent
- Figma 设计稿
- 设计稿还原的前端页面
- Supabase 数据驱动的完整应用
- 已部署上线的可访问产品
- 基于 Dify 的 AI 知识库应用
