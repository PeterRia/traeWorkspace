# 01 素材生成 Agent - 理论笔记

## 核心概念

### Lovart
AI 驱动的素材生成工具，用于快速创建 Logo、图标、插画等视觉素材。

### Nanobanana
轻量级 AI 绘图工具，适合快速生成产品所需的图片素材。

## 关键能力

1. **素材生产工作流**：从需求描述到素材产出
2. **绘图 Agent 搭建**：让 AI 自动管理素材生成流程
3. **批量素材生成**：一次生成多个风格统一的素材

## 实践要点

1. 用自然语言描述素材需求（风格、用途、尺寸）
2. 保持素材风格一致性
3. 建立素材管理和版本控制系统
