# 02 Figma 与 MasterGo - 理论笔记

## 为什么需要学习 UI 设计工具？

在 Vibe Coding 流程中，设计工具帮助你：
- 快速画出产品界面的低保真/高保真原型
- 与 AI 沟通时提供视觉参考
- 输出可直接转代码的设计标注

## Figma vs MasterGo

| 特性 | Figma | MasterGo |
|------|-------|----------|
| 开发公司 | Figma (美国) | 蓝湖 (中国) |
| 语言 | 英文为主 | 中文原生 |
| 免费版 | 有 | 有 |
| 协作 | 实时多人 | 实时多人 |
| 社区资源 | 极其丰富 | 快速增长 |
| 国内访问 | 可能较慢 | 快速稳定 |

## 基础操作速查

### 核心操作
- **Frame（画板）**：页面/屏幕的容器，按实际设备尺寸设置
- **Shape（形状）**：矩形、圆形、线条等基础图形
- **Text（文字）**：添加和编辑文本，设置字体大小和样式
- **Auto Layout（自动布局）**：元素自动排列和对齐
- **Components（组件）**：可复用的设计元素

### Vibe Coding 工作流
1. 用 AI 描述页面布局 → AI 给出设计建议
2. 在 Figma/MasterGo 中画出设计稿
3. 截图/导出标注 → AI 生成前端代码
4. 在浏览器中验证和微调
