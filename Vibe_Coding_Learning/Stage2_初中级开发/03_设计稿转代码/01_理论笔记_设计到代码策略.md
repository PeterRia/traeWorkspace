# 03 设计稿转代码 - 理论笔记

## 核心概念

将 Figma/MasterGo 设计稿转换为可在浏览器运行的前端代码（HTML/CSS/JavaScript），减少手工重搭的工作量。

## 转换策略

### AI 辅助转换
1. 截图 + 描述 → AI 生成 HTML/CSS
2. 设计标注 + 要求 → AI 生成 React/Vue 组件
3. 设计 token 导出 → 配合 Tailwind CSS 快速实现

### 提示词模板
```
根据以下设计稿描述，生成对应的前端代码：
- 布局：[描述布局]
- 配色：[描述配色方案]
- 组件：[列出需要的组件]
- 交互：[描述交互行为]
- 技术栈：[React/Vue/HTML]
```
