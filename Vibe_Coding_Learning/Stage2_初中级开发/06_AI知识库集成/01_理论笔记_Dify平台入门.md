# 06 AI 知识库集成 - 理论笔记：Dify 入门

## 什么是 Dify？

Dify 是一个开源的 LLM 应用开发平台，让非技术人员也能构建 AI 应用。

## 核心能力

### 知识库
- 上传文档（PDF、Word、网页等）
- 自动分段和向量化
- 混合检索（关键词 + 语义）

### 应用类型
- **对话型**：智能客服、问答助手（基于知识库精准回答）
- **文本生成型**：内容创作、摘要总结
- **Agent 型**：自动调用工具、执行多步任务

### 工作流编排
- 可视化拖拽构建 AI 流程
- 条件分支、循环、变量传递
- 连接外部 API 和工具

## RAG 基础（检索增强生成）

```
用户提问 → 知识库检索 → 获取相关文档 → LLM 整合回答 → 返回结果
```

### 为什么需要 RAG？
- LLM 的知识有截止日期
- LLM 不知道你的私有数据
- RAG 让 LLM 基于你的专属知识回答
