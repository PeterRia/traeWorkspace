# Stage 2 练习题与阶段总结汇总

## 01 素材生成 Agent - 练习题

1. 分别使用 Lovart 和 Nanobanana 生成你产品的 Logo，对比效果
2. 设计一套统一的素材风格规范（配色、字体、图标风格）
3. 用 AI 生成 5 个风格统一的图标

## 03 设计稿转代码 - 练习题

1. 选一个你喜欢的网站页面，截图为设计稿参考
2. 用 AI 将设计稿描述转换为 HTML/CSS 代码
3. 对比原页面和 AI 生成的效果，记录差异
4. 通过多轮对话让 AI 不断接近原设计

## 04 真实数据项目 - 练习题

1. 在 Supabase 上创建一个新项目
2. 设计用户表（id, name, email, created_at）
3. 配置行级安全策略（RLS）
4. 用 AI 生成前端代码连接 Supabase API

## 05 部署上线 - 练习题

1. 将任一项目部署到 Vercel
2. 配置自定义域名（可选）
3. 设置 HTTPS
4. 记录完整的部署步骤和遇到的问题

## 06 AI 知识库集成 - 练习题

1. 在 Dify 上创建一个知识库应用
2. 上传 3 篇文档作为知识库
3. 测试知识库检索的准确性
4. 将 AI 知识库集成到你的项目中
