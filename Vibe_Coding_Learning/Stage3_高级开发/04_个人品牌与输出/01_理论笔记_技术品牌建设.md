# 04 个人品牌与输出 - 理论笔记

## 为什么要建个人品牌？

- 学习成果可沉淀、可展示
- 让别人发现你的能力和作品
- 技术博客是最好的简历
- 长期价值积累（SEO、影响力）

## 个人品牌三件套

### 1. 个人网站
- 自我介绍和作品集展示
- 技术栈：Vibe Coding 快速搭建
- 推荐：Next.js + Vercel 部署

### 2. 技术博客
- 记录学习过程和技术心得
- 分享 Vibe Coding 实践经验
- SEO 优化让更多人看到

### 3. GitHub 主页
- 项目代码开源展示
- README 专业化
- Commit 记录展示学习过程

## 内容策略

| 内容类型 | 适合主题 | 频率 |
|---------|---------|:--:|
| 教程 | Vibe Coding 经验分享 | 每周 |
| 项目复盘 | 每个完成项目的总结 | 按项目 |
| 工具推荐 | 好用的 AI 工具 | 每月 |
| 思考感悟 | 技术趋势、学习方法 | 不定期 |
