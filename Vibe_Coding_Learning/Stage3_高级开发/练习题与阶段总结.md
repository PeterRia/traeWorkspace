# Stage 3 练习题与阶段总结汇总

## 01 跨平台桌面应用 - 练习题

1. 用 Vibe Coding 搭建一个简单的 Electron 应用骨架
2. 实现一个"语音转文字"功能（调用 AI API）
3. 测试在 Windows 上的运行效果

## 02 AI 智能体团队 - 练习题

1. 设计一个需要 3 个 Agent 协作的简单项目
2. 为每个 Agent 编写角色提示词
3. 记录 Agent 之间的协作效率和问题

## 03 长效稳定执行 - 练习题

1. 设计一个需要运行超过 30 分钟的任务
2. 实现断点续传机制
3. 测试任务中断后的恢复能力

## 04 个人品牌与输出 - 练习题

1. 用 Vibe Coding 搭建个人网站
2. 写一篇 Vibe Coding 学习经验博客
3. 优化网站的 SEO 和加载速度

## 综合项目建议

完成三个阶段后，建议做一个综合项目来检验学习成果：

### 项目要求
- 包含前端页面 + 后端 API + 数据库
- 集成了至少一种 AI 能力（对话/图像/知识库）
- 已部署上线，可通过 URL 访问
- 有完整的项目文档和用户使用说明

### 项目方向参考
- AI 问答助手（Dify 知识库 + Web 界面）
- AI 图片生成工具（AI API + 用户管理）
- 个人博客系统（Next.js + Supabase + AI 写作助手）
- 小游戏平台（多个 AI 游戏 + 排行榜）
