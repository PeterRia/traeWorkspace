# 博客复刻提示词

> 使用 web-clone-specialist 技能完整复刻 https://shw2018.github.io/ 博客网站到 my-blog 目录。

## 核心指令

请使用 web-clone-specialist 技能，1:1 复刻目标博客的所有页面和内容。技术栈采用 **Hexo 框架**配合 **hexo-theme-matery 主题**（Material Design 风格），开发环境需要 **Node.js 12.0+** 和 **npm 6.0+**。

## 技术栈详情

### 核心框架
- **Hexo**: 静态博客框架（基于Node.js）
- **hexo-theme-matery**: Material Design风格主题
- **Materialize CSS**: 前端UI框架
- **ECharts**: 图表库（用于统计图）

### 必需依赖
```
hexo, hexo-cli, hexo-server
hexo-generator-archive, hexo-generator-category
hexo-generator-index, hexo-generator-tag
hexo-renderer-ejs, hexo-renderer-marked
hexo-renderer-stylus, hexo-server
```

## 复刻范围

### 关键页面（必须100%还原）

#### 1. 首页 (/)
- **布局**: 导航栏 + Banner轮播 + 置顶文章轮播 + 推荐文章 + 文章列表 + 页脚
- **导航栏**: Logo(深度驿站) + 首页/标签/分类/归档/关于/留言板/友人帐/赞赏/搜索
- **Banner区域**: 10张轮播背景图 + 标题"深度驿站" + 副标题 + "开始阅读"按钮
- **置顶文章**: 4篇文章卡片自动轮播
- **推荐文章**: 4列卡片布局，展示4个分类的代表文章
- **文章列表**: 瀑布流式，水平卡片（左图右文），12篇文章
- **页脚**: 访问量统计 + 访客人数 + 字数统计(46.8k) + 版权信息

#### 2. 分类页 (/categories/)
- **Banner**: 背景图 + 标题"分类" + 诗句
- **文章分类**: 15个分类标签（网站优化、软件安装与配置、深度学习、计算机视觉等）
- **文章标签**: 46个标签展示
- **标签云**: 所有标签的可视化展示

#### 3. 标签页 (/tags/)
- **词云展示**: 标签根据文章数量动态调整大小
- **交互**: 点击标签跳转到对应文章列表

#### 4. 归档页 (/archives/)
- **时间轴布局**: 垂直时间轴
- **时间分组**: 按年 > 月 > 日分组（2019年10月/9月/8月）
- **文章信息**: 特色图片 + 标题 + 摘要 + 日期 + 分类 + 标签

#### 5. 关于页 (/about/)
- **个人信息卡片**: 头像 + 姓名 + 学校 + 专业 + 简介
- **统计信息**: 文章数(23) + 分类数(15) + 标签数(46)
- **文章统计图**: ECharts图表展示发布趋势
- **我的技能**: 进度条展示（算法80%、C++70%、Python95%、Java40%、HTML70%、CSS70%、JavaScript75%、NodeJS70%）
- **详细简历**: 个人概况 + 当前所在 + 专利/论文 + 科研/项目/比赛 + 主要获奖 + 工作实践

#### 6. 文章页 (/posts/*.html)
- **头部信息**: 标题 + 标签 + 分类 + 发布日期 + 文章作者 + 文章字数 + 阅读时长 + 阅读次数
- **文章内容**: Markdown渲染后的HTML内容
- **底部信息**: 文章作者 + 文章链接 + 标签列表
- **评论区**: Gitalk评论系统
