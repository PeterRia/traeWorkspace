# 玩家使用手册模板

> 该文件可直接作为 Codex 生成项目后的 `用户手册.md` 模板。

## 1. 游戏简介

《拼图冒险》是一款滑动拼图小游戏。玩家需要通过移动空白块旁边的拼块，把被打乱的图片恢复成原图。游戏包含多张主题图片、难度选择、原图参考、计时计步、提示、成就和本地记录。

## 2. 启动游戏

### 方式一：运行 exe

双击：

```text
dist/PuzzleGame/PuzzleGame.exe
```

或：

```text
dist/PuzzleGame.exe
```

### 方式二：运行源码

```bat
python -m pip install -r requirements.txt
python tools\generate_assets.py
python main.py
```

## 3. 主菜单说明

| 按钮 | 功能 |
|---|---|
| 开始游戏 | 选择图片、难度和模式后开始 |
| 图鉴 | 查看已解锁图片和完成记录 |
| 排行榜 | 查看不同难度的最佳成绩 |
| 设置 | 调整音效、原图提示、操作模式等 |
| 退出 | 关闭游戏 |

## 4. 难度说明

| 难度 | 棋盘 | 适合玩家 |
|---|---|---|
| 简单 | 3x3 | 第一次体验或快速通关 |
| 普通 | 4x4 | 推荐默认难度 |
| 困难 | 5x5 | 想挑战更高步数规划的玩家 |

旋转模式会让拼块方向也被打乱，需要额外旋转拼块，难度明显提升。

## 5. 操作说明

| 操作 | 效果 |
|---|---|
| 鼠标左键点击 | 移动与空白块相邻的拼块 |
| 鼠标拖拽 | 将相邻拼块拖到空白格，若已启用 |
| 鼠标右键 | 旋转拼块，旋转模式下有效 |
| H | 高亮所有可移动拼块 |
| G | 显示下一步建议 |
| Ctrl+Z | 撤销 |
| Ctrl+Y | 重做 |
| R | 旋转当前选中或悬停拼块 |
| M | 开关音效 |
| ESC | 暂停或返回 |

## 6. 游戏界面说明

游戏左侧是拼图区，右侧是信息栏。信息栏包含：

- 原图参考。
- 当前难度。
- 用时。
- 步数。
- 提示次数。
- 当前模式。
- 重开、提示、暂停、返回等按钮。

## 7. 提示与评级

提示可以帮助玩家找到可移动拼块或推荐下一步，但会影响最终评级。

最终评级参考：

- S：速度快、步数少、提示少。
- A：表现优秀。
- B：正常完成。
- C：完成但耗时或步数较多。

## 8. 图鉴与成就

初始只解锁第一张图片。完成当前图片后，会逐步解锁新的主题图片。每张图会记录：

- 完成次数。
- 最快完成时间。
- 最少步数。

成就会在满足条件时自动解锁，例如首次完成、快速完成、低步数完成等。

## 9. 成绩卡片

每次完成拼图后，游戏会在 `screenshots/` 文件夹生成一张成绩卡片，包含图片、难度、用时、步数、评级和完成日期。

## 10. 常见问题

### 游戏打不开

先尝试用源码运行：

```bat
python main.py
```

如果源码能运行但 exe 不能运行，通常是打包资源路径问题。

### 没有声音

游戏会在音频设备不可用时自动静音，不影响游玩。

### 存档丢失

检查 `saves/save_data.json` 是否存在。若文件损坏，游戏会自动备份并创建新存档。

### 中文显示异常

Windows 上通常使用微软雅黑或黑体。如果系统缺少中文字体，可能会出现方块字。
