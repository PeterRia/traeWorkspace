# 用户手册

## 启动游戏

双击 `run_game.bat`，或在命令行进入项目根目录后执行：

```bat
python main.py
```

首次运行会检查并生成缺失的图片、音效、存档目录。

## 基本玩法

1. 在主菜单点击“开始游戏”。
2. 选择一张图片。
3. 选择 3x3、4x4 或 5x5 难度。
4. 点击拼图区内与空白格上下左右相邻的拼块，该拼块会滑入空白格。
5. 点击空白格、非相邻拼块或拼图区以外的位置，不会改变棋盘。
6. 所有拼块回到正确位置后进入结算页。

## 游戏界面

右侧会实时显示原图参考、用时、步数、难度、提示次数和当前模式。顶部按钮支持重开、暂停、换图、换难度、提示、撤销、重做。

## 键盘操作

- `Esc`：暂停或返回
- `P`：暂停
- `R`：重新开局
- `H`：高亮当前可移动拼块
- `G`：显示曼哈顿距离提示
- `Ctrl+Z`：撤销
- `Ctrl+Y`：重做

## 图鉴和成就

通关会收录当前图片，并逐步解锁后续图鉴。成就包括首次通关、不使用提示、完成 5x5、快速完成 3x3、低步数完成和收集图鉴。

## 成绩卡

每次通关会在 `screenshots/` 中生成 `score_YYYYMMDD_HHMMSS.png`，包含图片名、难度、用时、步数、提示次数、评级和生成时间水印。

## 存档位置

源码运行时为 `saves/save_data.json`。exe 运行时为 `PuzzleGame.exe` 同级目录下的 `saves\save_data.json`。如果存档损坏，游戏会自动备份旧文件并创建新存档。
